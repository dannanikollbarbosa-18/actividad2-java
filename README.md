## Descripción de la actividad 2
Este proyecto es una aplicación básica en Java que representa un producto mediante una clase Product.
La clase incluye:

Atributos privados (id, name, price, stock)

Constructores (por defecto y con parámetros)

Métodos getters y setters con validación

Método toString() para mostrar la información del producto

El propósito del proyecto es practicar conceptos fundamentales de Programación Orientada a Objetos como encapsulamiento y uso de métodos.

## compilación y ejecutación
Le damos click a Run Java y este abre una terminal donde nos muestra los resultados
